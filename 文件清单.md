## 🚀 如何访问

### 在浏览器中访问
1. 启动项目：`npm start`
2. 访问演示页面：`http://localhost:3001/json-component-demo`
3. 或点击导航栏的"通用组件演示"按钮

### 在项目中使用
```jsx
import JsonTableDisplay from './components/JsonTableDisplay';

// 最简单的用法
<JsonTableDisplay data={yourJsonData} />
```
